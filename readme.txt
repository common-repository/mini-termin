Mini Terminkalender, jeder Beitrag kann zum Terminbeitrag werden

== Description ==
Mini Terminkalender, jeder Beitrag kann zum Terminbeitrag werden, mit Sidebarwidget, Shortcode, Eintragung im Content des Beitrages

== Installation ==


1. Installiere (lade) den Ordner `minitermin` in das `/wp-content/plugins/` Verzeichnis
1. Aktiviere das Plugin über das 'Plugins' Menü in WordPress
1. Shortcode [Termine] [termine cat='4']
1. Sidebarwidget mini Termin
